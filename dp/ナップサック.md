## 01-最小個数部分和

ある価値になる選び方のうち、個数が最小になるもの。

* 計算量: 時間 O(|A|ΣA), 空間 O(ΣA)
* 引数:
  * `A`: 非負整数の集合
* 返り値: 

```rust
fn subset_sum_min_count(A: &Vec<usize>) -> Vec<Option<usize>> {
  let sum = A.iter().sum::<usize>();
  let mut dp = vec![None; sum];
  dp[0] = Some(0);
  for &x in A {
    for y in x ..= sum {
      let c = dp[y - x] + 1;
      if dp[y].map(|d| d > c ).unwrap_or(true) {
        dp[y] = Some(c);
      }
    }
  }
  dp
}
```
